NNBuilder
===========
## Description
* A Neural Network Framework
* On Top Of Theano
## Dependency
* Python2.7
* Theano
* Matplotlib
* Progressbar
* Pydot Pydot-ng Graphviz
## Installation
### Using PIPY:
`pip install nnbuilder`
### For Theano Plaese:
`conda install theano pygpu`
`conda upgrade numpy`
### For Other Dependency:
`pip install pydot pydot-ng progressbar`
